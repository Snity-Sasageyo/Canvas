# Canvas

An endless drawing web application. Built with HTML, CSS, and JavaScript, this project fits the "Endless" theme by providing a space with absolutely no physical limits, edges, or boundaries.

## Features

* Endless: No edges, truely infinite(obv.)
* Tools: Pen and Eraser with customizable color and sizes
* History System: Undo or Redo functionality
* Export: Export your current design as a .png photo
